# Type


## データタイプ



### スカラー型

| 型             | 例                                             |
| -------------- | ---------------------------------------------- |
| 符号付き整数型   | `i8` `i16` `i32` `i64` `isize` (省略時は`i32`) |
| 符号なし整数型   | `u8` `u16` `u32` `u64` `usize`(省略時は`f64`)  |
| 浮動少数型      | `f32` `f64`                                    |
| 論理型          | `let t: bool = true;`(通常、型推論で省略可)    |
| 文字型          | `let c: char = 'x';`(通常、型推論で省略可)     |

<br/>

### 数値リテラル

| リテラル           | 例           |
| ------------------ | ------------ |
| 10進数             | `12_000`     |
| 16進数             | `0xff`       |
| 8進数              | `0o77`       |
| 2進数              | `0b1111_000` |
| バイト(`u8`型のみ) | `b'A'`       |
